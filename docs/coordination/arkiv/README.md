# Coordination-arkiv

Afsluttede plan-filer og udløbne arbejds-state-filer. Flyttes hertil når pakken er merget og slut-rapport er leveret.

Formålet er at holde `docs/coordination/`-rod konkret og fokuseret på igangværende arbejde, mens historikken bevares for audit-spor.

Filer her:

- Plan-filer fra afsluttede pakker (R-runde-2, R7h, m.fl.)
- Recon-filer fra afsluttede pakker

Slettes ikke. Henvises til ved retroaktive analyser.
