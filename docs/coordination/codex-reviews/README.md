# Codex-reviews

Output fra Codex-review-runder. Én fil pr. review-runde, navngivet `<dato>-<pakke-kode>-runde-<N>.md`.

Indhold pr. fil:

- Codex' rå output
- Code's modtagelses-summary (niveau 3 fra `docs/skabeloner/codex-review-prompt.md`)
- Mathias' afgørelser pr. fund

Append-only struktur. Reviews flyttes ikke til arkiv — historikken bevares her som audit-spor for hvordan beslutninger blev modnet.
